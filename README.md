Wanderlust - Full Stack Vacation Rental Platform
Wanderlust is a professional-grade, full-stack web application inspired by Airbnb. It allows users to list, discover, and book unique accommodations worldwide. This project was built to demonstrate proficiency in the MVC (Model-View-Controller) architecture, secure authentication, and cloud-based image management.
